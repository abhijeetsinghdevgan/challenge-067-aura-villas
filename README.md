# Aura Villas - Luxury Vacation Rental Booking

Welcome to the **Aura Villas** project! This is a premium web interface designed for a luxury vacation property booking system, showcasing a stunning dark-mode aesthetic, responsive design, and an interactive booking calculation system.

## 🌟 Features

*   **Premium Aesthetic**: Dark mode theme with Champagne Gold accents, glassmorphism elements, and modern typography (Playfair Display & Inter).
*   **Interactive Booking Form**:
    *   Dynamic date validation (Check-out must be after Check-in, minimum date is today).
    *   Real-time cost calculation based on selected dates, suite type, and extra preferences.
    *   Inclusion of essential elements like non-smoking requirements and security deposit information.
*   **Responsive Layout**: Fully responsive design ensuring a seamless experience across desktop, tablet, and mobile devices.
*   **Smooth Animations**: Scroll-based fade-up animations and hover effects.
*   **Success Modal**: A clean modal confirming the reservation upon form submission.

## 🛠️ Technologies Used

*   **HTML5**: Semantic markup for content structure.
*   **CSS3**: Custom variables, Flexbox, CSS Grid, media queries, and smooth animations (No external frameworks).
*   **JavaScript (Vanilla)**: DOM manipulation, date logic, price calculation, and event handling.
*   **Lucide Icons**: Beautiful, clean SVG icons.

## 📝 Design Decisions

*   **Color Palette**: Chose a deep navy/slate background (`#0A0D14`) paired with a gold accent (`#D4AF37`) to convey luxury, exclusivity, and tranquility.
*   **Typography**: *Playfair Display* is used for headings to give a high-end, editorial feel, while *Inter* provides excellent readability for the UI and body copy.
*   **User Experience**: The booking form is designed to be clear and intuitive, providing immediate feedback on pricing to build trust with the user.
